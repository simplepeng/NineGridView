# NineGridView

可能是最通用的九宫格布局

* 正常九宫格视图，仿微信，Item显示样式可完全自定义
* 超过9个Item，可自定义额外展示的布局
* 自定义适配单个item视图显示样式
* 自定义item间的间距
* 适配4个item时的显示样式，仿B站
